# Playwright-Personal-Project
Project automation testing by Playwright + TypeScript using Page Object Model (POM)

## **Project Structure**  
PLAYWRIGHT-PERSONAL-PROJECT
│── Data/                 # Test data  
│── Fixtures/             # Fixtures & reusable setup  
│── Pages/                # Page Object Model classes  
│── playwright/           # Playwright generated files  
│── playwright-report/    # HTML reports  
│── test-results/         # Test artifacts & screenshots  
│── tests/                # Test cases  
│── .env                  # Environment variables  
│── playwright.config.ts  # Playwright configuration  
│── package.json  
│── tsconfig.json  
<br>
**Tech Stack**  
>Playwright  
>TypeScript  
>Node.js  
>Page Object Model (POM)
